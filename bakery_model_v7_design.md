## 二、核心计算公式 (v7)

### PhysCap — 物理产能上限

```
physCap = min( area × 30,  staff × 1800 )
B3 = round( physCap × sk(FAT) )
```

- **工资不影响物理产能。** 高薪的作用体现为降低废品率（B4）和提升品质（B28），而非让人揉面更快。
- 60m²+3人 → min(1800,5400) = 1800 件
- 150m²+3人 → min(4500,5400) = 4500 件
- 200m²+8人 → min(6000,14400) = 6000 件

### sk(FAT) — 疲劳产能系数

```
FAT < 80:   1.0
FAT ≥ 100:  0.1
80–100:     sigmoid t³型平滑衰减, t = (FAT-80)/20
            sk = 1 - t²(3-2t)×0.9
```

### B4 — 加工成本（废品率）

```
wfB4 = 1.20 - 0.45 × min(工资, 10000) / 10000
B4 = max(0.03,  (1.5 - B3 × 0.00015) × wfB4 × (1 - EMP × 0.002) )
```

| 工资 | wfB4 | B3=2000 时 B4 |
|------|------|---------------|
| ¥800 | 1.16 | ¥1.39 |
| ¥4000 | 1.02 | ¥1.22 |
| ¥10000 | 0.75 | ¥0.90 |

- 高薪烘焙师 → 低废品率 → 加工成本下降。物理速度不变。
- 经验 (EMP) 也能降低 B4，上限 200 时额外 40% 减幅。

### B28 — 原料品质

```
qBase = clamp(0.15, 0.95,  B10 / 5.5 )
qSalary = B26 > 3000 ? min(0.18, (B26-3000)/28000) : 0
B28 = clamp(0, 1.0,  qBase + qSalary )
```

- 原料 ¥3 → 品质 0.55；¥6 → 0.95
- 高薪吸引好师傅，品质最多额外 +0.18
- 不再随产能（B3）自动提升品质——供应商折扣不改变原料档次

### B9 — 人工成本

```
B9 = round( 员工 × 工资 × (1 + 地段 × 0.10 × max(0, 1-员工×0.05)) )
```

- 地段乘数从 0.15 降为 0.10，管理衰减从 0.08 降为 0.05

### B5 — 租金（平滑衰减）

```
rate = 20 + 800 / (面积 + 15)
B5 = round( 面积 × 地段 × rate )
```

| 面积 | rate | 地段3 月租 | 地段8 月租 |
|------|------|-----------|-----------|
| 30m² | ¥37.8 | ¥3,402 | ¥9,072 |
| 60m² | ¥30.7 | ¥5,526 | ¥14,736 |
| 100m² | ¥27.0 | ¥8,088 | ¥21,568 |
| 200m² | ¥23.7 | ¥14,232 | ¥37,952 |
| 300m² | ¥22.5 | ¥20,280 | ¥54,080 |

- 反比例衰减，不会跳水也不会倒挂

### B12 — 实际原料成本（批量折扣）

```
B12 = round( B10 × (1 - min(0.4,  B3 × 0.00006)) × 100 ) / 100
```

上限折扣 40%（原版 50%），折扣速率降低。

### B2 — 零售客流需求（V7 核心改动）

**客流不再受价格影响。** 价格通过转化率筛选，而非凭空改变路过人数。

```
footTraffic = 500 + 400 × min(地段, 6)           ← 地段自然流
            + round(√面积 × 15)                   ← 店面可见度
            + round(营销^0.45 × 0.8)              ← 营销带客
            + round(品牌 × 0.8)                    ← 口碑

社区密度加成: 地段3-5 且 面积≤90 → ft × 1.15
```

**品牌溢价（对数衰减）**

```
brandP = round( min(45,  12×ln(1+BRAND/25) × min(1, (grade-1)/8) ) )
```

| BRAND | 地段1 | 地段5 | 地段9 |
|-------|-------|-------|-------|
| 25 | 0 | 8 | 17 |
| 100 | 0 | 14 | 26 |
| 200 | 0 | 16 | 26 |

- 低地段（<5）品牌溢价严重受限——好面包在工业区卖不出奢侈品价

**最大可接受价 ma**

```
ma = max( 12 + 地段×2.5 ,  12 + 地段×2.5 + brandP + B21×2 + B28×3 + 工资/2500 )
```

**价格弹性（仅作用在转化率）**

```
pbF(price):
  price < 15:   1 + (15-price)×0.10     ← 低价刺激
  price ≤ 22:   1.0                      ← 正常区间
  price > 22:   max(0.40, 1-(price-22)×0.03)  ← ¥38时 0.52
```

**软转化衰减（无断崖）**

```
gap = max(0, B1 - ma)
conv = 0.65 × pbF(B1) × ( ma / (ma + gap×0.8) )
conv = clamp(0.04, 0.82, conv)
```

即使价格超过 ma，也只是平滑下降——不会突然从 0.8 跳到 0.05。

**季节 + 品牌护城河**

```
SZ = [0.85, 1.10, 0.95, 1.00, 1.00, 0.90, 0.85, 0.85, 1.25, 1.10, 1.00, 1.30]

loyalSz = BRAND≥200 ? min(sz,1.08) : (BRAND≥100 ? min(sz,1.12) : sz)
finalSz = max(loyalSz,  BRAND≥200?0.95 : (BRAND≥100?0.90 : sz))
```

**旅游客流**（地段≥5 触发）

```
tourBase = (B15-4) × 500 × pbF(B1)
tourMkt  = round(√营销 × 5) × (0.5 + BRAND/400)
tourMa   = max(8, 8 + B15×2 + 营销/4000 + BRAND×0.25)
tourConv = B1≤tourMa ? min(0.75, 0.30 + (tourMa-B1)/tourMa×0.35) : max(0.02, 0.25×tourMa/B1)
tourD    = round( (tourBase + tourMkt) × tourConv × sz )
```

```
B2 = localDemand + tourDemand
```

### B6 — 实际销量

```
B6 = min(B2, B3)
```

### B7 — 月营收

```
零售收入 = B6 × B1                              ← 售价即客单价，无隐形+20%
B2B 收入 (面积≥150 触发):
  b2bCap = 员工 × 3000                           ← 每人月接3000件企业订单
  spotCap = round((面积-150) × 500)               ← 冗余产能走批发行情
  wsSold = min(b2bCap+spotCap,  B3-B6)
  wsRev  = round(wsSold × B1 × 0.30)             ← 批发价=零售30%

B7 = 零售收入 + wsRev
```

### B8 — 月利润

```
零售 COGS = round( (B12 + B1×0.10 + B4) × 零售销量 )   ← 包装费10%
B2B COGS  = round( wsSold × B10×0.50 + wsSold×0.3 )     ← B2B原料5折+简包装
固定成本:
  租金    = B5
  人工    = B9
  营销    = B13
  培训    = B25 × B24
  杂费    = round(0.02 × B24 × 1000)
  水电折旧 = round(B14 × 18 + B24 × 120) + 1200

B8 = B7 - 零售COGS - B2BCOGS - 所有固定成本
```

### B20 — 口味/品质

```
B20 = round( B21 × 0.45 + B28 × 0.55, 2 )
```

## 三、状态演进（每月推进变化）

### FAT (疲劳度 0–100)

```
零售 UR = min(1,  B2 / physCap )    ← 只跟零售需求挂钩，B2B不产生疲劳

FAT 变化 d:
  UR > 88%:  d = round((UR-0.88)×35)
  UR > 95%:  d += 2
  UR < 75%:  d = -round((0.75-UR)×15)
  面积>150 且 人数 < ceil(面积/35):  d += 3    ← 人手不足惩罚

newFAT = clamp(10, 100, FAT + d)
```

### BRAND (知名度)

```
growth = B21×8 + B28×10 + max(0,(100-FAT)/100)×5 + √营销×0.12
decay  = BRAND × 0.015
gm     = max(0.03,  1 - BRAND/500)
BRAND += growth×gm - decay - 缺货惩罚( sr × 8 )

品质惩罚 (B28<0.40): 装修扣 gap×20, 天花板 25-50
```

### EMP (经验值 0–200)

```
EMP += max(1, 10 - round(EMP×0.05))
```

### B21 (员工满意度 0–1)

```
pp = B9 / max(B3,1)        ← 边际人工成本
bl = 2.5 + B15×0.3         ← 地段基准线
ps = pp≥bl ? 0.7+min((pp-bl)/(bl×2),0.25) : pp/bl×0.7
ov = max(0, B3/physCap - (0.8+0.2×ps)) × 1.2
B21 = clamp(0,1, ps - ov)

FAT 侵蚀: FAT>60 → -0.004×(FAT-60), FAT>85 → -0.010×(FAT-85)
```
