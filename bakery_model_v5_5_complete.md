## 二、用户滑条 (3组决策面板)

**👥 人事**
- 人数 B24: 1~20, step 1, 默认 8
- 工资 B26: ¥800~10,000, step 200, 默认 5,000
- 培训 B25: ¥0~1,000, step 50, 默认 100

**📢 市场**
- 售价 B1: ¥5~40, step 1, 默认 16
- 营销 B13: ¥0~20,000, step 500, 默认 2,000

**🥖 供应链**
- 原料 B10: ¥1~6, step 0.1, 默认 5
- 面积 B14: 30~300㎡, step 10, 默认 150
- 地段 B15: 1~10, step 1, 默认 7

## 三、单月计算流程

### 3.1 人工成本 B9

```
复杂度系数 = 1 + B15 × 0.15 × max(0, 1 - B24 × 0.08)
  → 地段越好、人越少 → 人均成本越高(需要更专业的员工)

B9 = round(B24 × B26 × 复杂度系数)
```

例: B15=7, B24=8, B26=5000 → B9 = 8×5000×(1+7×0.15×0.36) = 40,000×1.378 = 55,120

### 3.2 工资正反馈

#### a) 工资→产能系数 wf

基准: ¥5,000/人 → 1.0

```
wf = 0.2 + 1.6 × B26 / 10,000
```

| B26 | wf |
|:---:|:--:|
| ¥3,000 | 0.68 |
| ¥5,000 | 1.00 |
| ¥8,000 | 1.48 |
| ¥10,000 | 1.80 |

#### b) 工资→报废系数 waste_factor

基准: ¥5,000/人 → 0.50

```
waste_factor = max(0.3, 1.5 - B26 / 5,000)
```

| B26 | waste_factor |
|:---:|:-----------:|
| ¥3,000 | 0.90 |
| ¥5,000 | 0.50 |
| ¥8,000 | 0.30 |
| ¥10,000 | 0.30 |

#### c) 工资→品质溢价 quality_premium

高薪师傅手艺更好, 顾客愿意多出钱

```
quality_premium = max(0, (B26 - 4,000) / 500)
```

| B26 | ma 增加 |
|:---:|:-------:|
| ≤¥4,000 | 0 |
| ¥5,000 | +2 |
| ¥8,000 | +8 |
| ¥10,000 | +12 |

### 3.3 产能与产量 B3

```
面积产能 = B14 × 35 (35份/㎡/月)
人效产能 = B24 × 1,500 (1,500份/人/月 ≈ 50份/天)
产能恢复 = max(0, (2 - B4) × 100)

ph = max(1, min(面积产能, 人效产能) × wf + 产能恢复)
B3 = round(ph × sk(FAT))
```

疲劳系数 sk(c4):
- c4 < 80: 1.0
- 80 ≤ c4 < 90: 0.30
- c4 ≥ 90: 0.10

### 3.4 报废成本 B4

```
B4 = max(0.1,
         max(0.1, 2 - B3×0.0002) × (1 - EMP×0.002) × waste_factor)
```

### 3.5 价格弹性 pb

```
pb = 1 + (20 - B1) × 0.15    当 B1 < 20 (低价引流)
     max(0.6, 1 - (B1-20)×0.03)    当 B1 ≥ 20 (高价赶人)
```

| B1 | pb |
|:--:|:--:|
| ¥15 | 1.75 (大幅引流) |
| ¥20 | 1.00 (持平) |
| ¥30 | 0.70 (客流打7折) |
| ¥40 | 0.60 (下限) |

### 3.6 客流 TRAFFIC

```
base_tr = round(500 + 500×B15) + round(BRAND × 3)
  → B15=3 时 2,000人/月 (67人/天)
  → B15=7 时 4,000人/月 (133人/天)
  → B15=9 时 5,000人/月 (167人/天)
  → 品牌100 时 +300人/月

area_lever = 1 + B14 / 100
mkt_tr = round(√max(0, B13) × 10)
  → 营销¥8,000 → √8000×10 ≈ 894人

TRAFFIC = round(base_tr × pb) + round(mkt_tr × area_lever)
```

### 3.7 需求与转化 B2

```
ma = max(1, 15 + B15×2 + BRAND×0.5 + B21×3 + quality_premium)
  → B15=3 时 ma≈21 (社区顾客穷)
  → B15=9 时 ma≈33 (商场顾客富)
  → 品质溢价 +8 (高薪师傅做得好吃)
```

转化率:
```
若 B1 ≤ ma:  conv = 0.5 + (ma - B1)/ma × 0.4
若 B1 > ma:  conv = max(0.05, 0.5 × ma / B1)
```

例: ma=33, B1=30 → conv = 0.5+(33-30)/33×0.4 = 53.6%
    ma=33, B1=35 → conv = max(0.05, 0.5×33/35) = 47.1%

```
B2 = round(TRAFFIC × conv)
```

### 3.8 房租 B5

```
B5 = max(0, round(B14 × B15 × max(2, 20 - B14×0.05)))
```

| 配置 | 月租金 |
|:----|:-----:|
| 60㎡, B15=3 | ¥3,060 |
| 100㎡, B15=9 | ¥13,500 |
| 250㎡, B15=7 | ¥13,125 |
| 300㎡, B15=10 | ¥15,000 |

### 3.9 员工满意度 B21

```
pp = B9 / max(B3, 1)                     (人均产值)
bl = 3 + B15 × 0.4                        (薪酬基准线)
```

薪酬满意度:
```
若 pp < bl:  ps = pp/bl × 0.7
若 pp ≥ bl:  ps = 0.7 + min((pp-bl)/(bl×2), 0.3)
```

```
超额惩罚 = max(0, B3/ph - (0.8 + 0.2×ps)) × 1.5
B21 = round(min(1, max(0, ps - 超额惩罚)) × 1000) / 1000
```

忍耐阈值 = 0.8 + 0.2×ps (高薪提高忍耐力)
超额惩罚: 产能利用率超过忍耐阈值的部分 × 1.5

### 3.10 原料成本 B12

```
数量折扣 = min(0.5, B3 × 0.00008)
B12 = round(B10 × (1 - 数量折扣) × 100) / 100
```

例: B10=5, B3=2,550 → B12 = 5×(1-0.204) = 3.98

### 3.11 销量与营收

```
B6 = min(B2, B3)                         (需求 vs 产能)

饮品收入    = round(B6 × B1 × 0.20)       (面包营收的20%)
B7 = B6 × B1 + 饮品收入                    (总营收)
```

### 3.12 成本汇总与利润 B8

```
包装成本  = round(B1 × 0.15)               (售价的15%)
总生产成本 = (B12 + 包装成本 + B4) × B3

杂项福利  = round(0.05 × B24 × 1,500)     (社保/杂费)
水电燃气  = round(B14 × 25 + B24 × 200)
设备折旧  = 2,000                          (烤箱/冰柜)

B8 = round(B7 - 总生产成本
            - B5 - B9 - B13
            - B25×B24 - 杂项福利
            - 水电燃气 - 设备折旧)
```

成本结构例 (高奢溢价, 稳态):

| 项目 | 金额 | 占比 |
|:----|:---:|:---:|
| 营收(面包+饮品) | ¥106,308 | 100% |
| 原料+包装+报废 | ¥37,208 | 35% |
| 房租 | ¥13,500 | 13% |
| 人工 | ¥33,000 | 31% |
| 营销 | ¥5,000 | 5% |
| 水电+折旧 | ¥5,300 | 5% |
| 其他 | ¥700 | 1% |
| **净利** | **¥22,000** | **21%** |

### 3.13 疲劳变化 fd()

```
d = 3                         (基线: 上班就有精神消耗)

if 培训=0 且 产能利用率>70%:  d += 5
if 培训=0:                    d += 2
if 产能利用率>80%:            d += (利用率-0.8)×40
d -= 培训费 × 0.03

if 满意度<0.5:  d -= (满意度-0.5)×15 × fatigue_factor
if 人均工资>1,500: d -= (人均工资-1500)×0.005 × fatigue_factor

fatigue_factor = FAT/40    当 FAT < 40
                 1         当 FAT ≥ 40

FAT = max(0, min(100, round(FAT + d)))
```

### 3.14 跨月更新 nx()

```
利润 = B8

经验(边际递减):
  EMP = min(200, round(EMP + max(1, 10 - EMP×0.05)))
  → 新员工学得快(+10), 老员工学得慢(+1)

品牌:
  缺货率    = max(0, (B2-B3)/max(1,B2))
  品牌增长  = B21 × 30
  品牌衰减  = BRAND × max(0.02, BRAND×0.015)
  增长系数  = max(0.1, 1 - BRAND/800)     (天花板800)
  BRAND = max(0, round(BRAND + 品牌增长×增长系数 - 品牌衰减 - 缺货率×10))

FAT = nf
```
